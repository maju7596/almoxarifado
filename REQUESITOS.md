
#  do Sistema de Almoxarifado
## Requesitos Funcionais
1. Cadastro de usuários com níveis de acesso (administrador, almoxarife, gestor e solicitante).
2. Autenticação de usuários por login e senha.
3. Cadastro de produtos e materiais.
4. Cadastro de fornecedores.
5. Registro de entrada de materiais no estoque.
6. Registro de saída de materiais do estoque.
7. Controle de estoque mínimo e máximo.
8. Consulta de saldo disponível em estoque.
9. Emissão de relatórios de movimentação de estoque.
10. Histórico de entradas e saídas de materiais.

## Requesitos Não Funcionais

1. O tempo de resposta das consultas deve ser inferior a 3 segundos.
2. O sistema deve possuir interface amigável e intuitiva.
3. Os dados devem ser armazenados de forma segura e criptografada.
4. O sistema deve permitir acesso simultâneo de múltiplos usuários.
5. O sistema deve ser compatível com navegadores modernos.
6. O sistema deve possuir controle de backup automático diário.
7. O sistema deve garantir rastreabilidade das operações.
